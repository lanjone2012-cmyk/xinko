# XINKO Space 上线检查清单

## 已准备

- 首页：`index.html`
- 三个产品落地页：`hub-01.html`、`rack-01.html`、`workstation-01.html`
- 五篇内容营销文章
- 感谢页：`thanks.html`
- 本地线索台账：`inquiries.html`
- SEO 文件：`sitemap.xml`、`robots.txt`
- Web App 清单：`site.webmanifest`
- 404 页面：`404.html`
- 静态部署配置：`_headers`、`_redirects`
- 部署说明：`部署说明.md`

## 上线前需要替换

- 把 `sitemap.xml` 和 `robots.txt` 里的 `https://example.com` 换成正式域名。
- 询盘接收渠道优先使用飞书；参考 `../INTEGRATIONS/飞书微信询盘接入方案.md`。
- 把 `config.js` 里的 `window.XINKO_INQUIRY_ENDPOINT` 改成正式后端接口地址。
- 若接入真实后端，保留本地台账作为备份或测试模式。

## 推荐部署方式

1. 使用 Vercel、Netlify、Cloudflare Pages 或静态服务器部署整个 `WEBSITE` 目录。
2. 绑定正式域名。
3. 提交站点地图到搜索引擎。
4. 用真实手机提交一次询盘，确认线索能进入目标渠道。
